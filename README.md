# DarkLight-uTools

> 从 uTools 切换 macOS 系统外观

这个是 [DarkLight](https://github.com/L1cardo/DarkLight) 的 [uTools](https://www.u.tools/) 插件版本，仅支持 macOS 系统

可以直接从 uTools 来切换 macOS 的 **深色模式** 与 **浅色模式**

本人学医，不会 js，超猫画虎复制粘贴搞了这么一个小插件，不喜勿喷
